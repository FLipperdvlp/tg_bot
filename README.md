# Telegram Bot + Playwright Session Automation

## 📌 Описание проекта

Данный проект представляет собой систему автоматизации авторизации на веб-сайтах с использованием **Telegram-бота** и библиотеки **Playwright (Python)**.

Основная цель проекта — упростить процесс входа в аккаунты по номеру телефона и автоматизировать получение сессионных данных (token / cookies) для последующего быстрого доступа без повторной авторизации.

---

## ⚙️ Принцип работы

Система реализует следующий сценарий:

1. Пользователь взаимодействует с Telegram-ботом.
2. Отправляет номер телефона в чат с ботом.
3. Бот запускает браузер через Playwright и открывает целевой сайт.
4. Автоматически вводится номер телефона в форму входа.
5. Пользователь получает код подтверждения (SMS / OTP).
6. Пользователь отправляет код в Telegram-бот.
7. Бот передаёт код в браузер и завершает процесс авторизации.
8. После успешного входа система извлекает данные сессии (session token / cookies).
9. Сессионные данные сохраняются для последующего повторного использования без необходимости повторного входа.

---

## 🧠 Архитектура проекта

Проект состоит из двух основных модулей:

### Telegram Bot
- Приём номера телефона от пользователя  
- Приём кода подтверждения (OTP)  
- Управление процессом авторизации  
- Передача данных в браузерный модуль  

### Playwright Automation
- Запуск и управление браузером  
- Автоматическое заполнение форм на сайте  
- Обработка логина  
- Извлечение сессионных данных (cookies / tokens)  

---

## 🔐 Работа с сессиями

После успешной авторизации система сохраняет данные сессии пользователя:

- session token  
- cookies  
- localStorage (при необходимости)

Эти данные используются для:

- быстрого входа в аккаунт без повторной авторизации  
- автоматизации последующих действий  
- восстановления сессии
